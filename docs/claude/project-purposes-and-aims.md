 
# 1. Purpose of the Project

The Building Materials Marketplace is a modular system that allows:

- Sellers/companies to manage stock and product specifications
- Customers to browse, search, filter, and order building materials
- Administrators to manage categories, specs, units, cities, and sellers

Primary goals:
- Scalable, modular architecture
- Fast product search and filtering
- Two-side Filament panel (Admin + Seller)
- Modern SPA customer experience
- Strict safety around automated code modification (Claude)

---

# 2. Aims of the Project

1. Build an enterprise-ready architecture with clear domain separation
2. Provide clean user experience for customers, sellers, and admins
3. Use API-first backend design
4. Use Vue 3 + Pinia for a modern SPA
5. Use Filament 4 for Admin and Seller dashboards
6. Ensure data consistency via category-based specification system
7. Protect the project from accidental tool-driven code changes (Claude rules)
